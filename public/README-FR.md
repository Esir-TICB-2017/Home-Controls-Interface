# Home-Controls-Interface
Projet étudiant domotique

Attention : l'application en est à ses tout débuts, la livraison d'une version beta fonctionnelle est prévue pour
fin Janvier/début Février. 

Le projet, développé dans un module scolaire en 4ème année d'école d'ingénieur a pour but d'offrir aux habitants ou utilisateurs 
d'un bâtiment un panneau de contrôle de l'habitat ainsi qu'une interface affichant les données reléves dans le bâtiment. 

Les contraintes que nous nous sommes fixées sont la possibilité d'utiliser l'application sur des appareils très différents
comme des ordinateurs, tablettes, smartphones. Ainsi l'application suit le responsive design. 

La première étape du développement est la mise en place de l'interface utilisateur. Par la suite, nous ajouterons la
connexion à la maison et aux données et contrôles réels.
